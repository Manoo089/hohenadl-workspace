# Manuel Hohenadl — Portfolio

> Ein One-Pager-Portfolio, der sich selbst ernstnimmt — aber nicht zu sehr.

Live: [hohenadl.dev](https://hohenadl.dev)

---

## Was ist das hier?

Portfolio eines Web Developers aus Bayern – gebaut mit Next.js 16. Denn reines HTML in einer `index.html` abzulegen gilt ja inzwischen als Persönlichkeitsstörung.

## Tech Stack

| Was                 | Womit                                      |
| ------------------- | ------------------------------------------ |
| Framework           | Next.js 16 (App Router)                    |
| Sprache             | TypeScript                                 |
| Styling             | SCSS (kein Tailwind — ich lebe gefährlich) |
| Fonts               | Syne + DM Mono                             |
| Syntax Highlighting | Shiki                                      |
| Deploy              | Vercel                                     |
